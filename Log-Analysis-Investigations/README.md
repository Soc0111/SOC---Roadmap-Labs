# 🔍 Cyber Incident Investigation Lab

## 📖 Project Overview
Welcome to my **Log Analysis Lab**! 🛡️ This repository serves as a professional portfolio of my technical investigations into various cyber threats. 

Here, I perform deep-dives into raw system logs to:
* 🕵️‍♂️ **Detect** malicious activity and anomalies.
* 🛡️ **Perform** digital forensics on compromised systems.
* 📝 **Document** comprehensive incident response steps and mitigation strategies.
